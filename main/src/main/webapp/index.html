Warning: servlet startup is excruciatingly slow on the dev server, but once loaded, page refresh is pretty fast.

Editing

You'll need to run the `fswatch.sh` script in each module subproject. That will launch some background processes to monitor the source tree and copy changed files to the appropriate module runtime tree at `ear/exploded-app`.

Then edit a source file and refresh the webpage. You should see the changes almost instantly.
